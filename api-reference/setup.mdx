## Configuration

### 1. Get Your API Key

1. Create an account at [Push Dashboard](https://dashboard.push.com)
2. Navigate to Settings > API Keys
3. Create a new API key
4. Copy your API key

### 2. Configure Your Backend

<Tabs>
  <Tab title="Node.js">
    ```javascript
    // Initialize the client
    const pushApi = require('@push/api');

    // Configure with your API key
    pushApi.configure({
      apiKey: process.env.PUSH_API_KEY
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    from push_api import PushAPI

    # Initialize the client
    push_api = PushAPI(api_key=os.getenv('PUSH_API_KEY'))
    ```
  </Tab>
  <Tab title="Ruby">
    ```ruby
    require 'push-api'

    # Initialize the client
    PushAPI.configure do |config|
      config.api_key = ENV['PUSH_API_KEY']
    end
    ```
  </Tab>
</Tabs>

### 3. Set Up Environment Variables

Add your API key to your environment variables:

```bash
# .env file
PUSH_API_KEY=your_api_key_here
```

<Note>
  Never commit your API key to version control. Always use environment variables.
</Note>
